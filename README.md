# 2048 Game Website

Welcome to the 2048 Game Website, hosted at [taylorswift2048.club](http://taylorswift2048.club). This website features multiple variants of the classic 2048 puzzle game, including the original version, Taylor Swift-themed version, Cupcakes version, and AI-powered version.

## Features

### Game Variants
- **Classic 2048**: The original 4x4 grid game that started it all
- **2048 Cupcakes**: Sweet version with cupcake-themed tiles and special features
- **Taylor Swift 2048**: Merge Taylor Swift's album covers to unlock new eras
- **AI 2048**: Watch an AI automatically play 2048 and reach the highest tile possible

### Additional Features
- Responsive design for all devices
- Sound effects (toggleable)
- Move counter and time tracking for Cupcakes variant
- SEO-friendly structure
- Clean, modern interface

## How to Play

1. **Classic 2048**: Use arrow keys to move tiles. Merge tiles of the same value to create larger numbers.
2. **2048 Cupcakes**: Same gameplay, but with cupcake-themed tiles and an undo feature.
3. **Taylor Swift 2048**: Merge album covers to progress through Taylor's musical eras.
4. **AI 2048**: Watch the AI play automatically, or adjust settings to experiment with different strategies.

## How to Run Locally

1. Clone this repository:
   ```bash
   git clone https://github.com/yourusername/taylorswift2048.git
   ```

2. Navigate to the project directory:
   ```bash
   cd taylorswift2048
   ```

3. Start a local server:
   ```bash
   python3 -m http.server 8080
   ```

4. Open your browser and visit `http://localhost:8080`

## Deployment

The website is deployed at [taylorswift2048.club](http://taylorswift2048.club). To deploy your own version:

1. Push the code to a GitHub repository
2. Use GitHub Pages, Netlify, or Vercel to deploy the static site
3. Configure your custom domain (e.g., taylorswift2048.club) in the hosting service
